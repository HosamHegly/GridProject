# GridProject: QA Testing Framework for 365Scores Website
## Overview
GridProject is a comprehensive QA testing solution designed for ensuring the functionality and navigability of the 365Scores website across various browsers. Utilizing the power of Selenium Grid, this project allows for both parallel and serial execution of tests, adaptable through user configurations.

## Features
Cross-Browser Testing: Execute tests on multiple browsers to ensure wide compatibility.
Parallel and Serial Execution: Customize test execution mode via the configuration file for efficiency and convenience.
Comprehensive Test Coverage: Includes tests for crucial website navigation aspects, enhancing user experience and reliability.
Getting Started
Ensure you have Selenium Grid set up and Python installed. For installation and running instructions, visit the infra directory.
